Aby odpalić aplikację należy:
1) Skopiować repozyotria na swój lokalny dysk
2) Skonfigurować bazę danych wg pliku application.properties
3) zbudować projekt
4) Urouchomić obie aplikację 
5) Wejść na http://localhost:8080
